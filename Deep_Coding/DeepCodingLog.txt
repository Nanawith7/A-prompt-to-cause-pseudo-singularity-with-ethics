    MDN Web Docs: Canvas API – 基本的な描画方法

    MDN Web Docs: KeyboardEvent – キー入力処理

    MDN Web Docs: requestAnimationFrame – ゲームループ実装

主要設計ポイント

    ゲームループ: requestAnimationFrame を使用し、毎フレーム更新→描画を行う。

    キー入力管理: keydown / keyup イベントでフラグ管理（上下左右、Z、X、Q）。Qキーのみトグル処理。

    状態管理: gameState 変数で 'title' / 'playing' / 'pause' / 'gameOver' / 'clear' を保持。

    自機: 三角形のポリゴン描画。移動範囲はCanvas内に制限。

    背景スクロール: 単純な横縞模様のオフセットを毎フレーム移動させる。

    ポーズ機能: 'playing' 時のみ更新処理を実行。Qキーで 'pause' ↔ 'playing' トグル。ポーズ中は画面中央に「PAUSE」表示。

3. 推論ステップと反証的検証
ステップ1: キー入力の同時押し対応

推論: 上下左右同時押しでも斜め移動ができるよう、各キーのフラグを独立に管理する。
反証的検証: keydown でフラグON、keyup でOFF。同じキーが連続イベントで多重にON/OFFされても問題ないが、ブラウザによってはリピート発火があるため、更新ループ内でフラグに基づき移動量を計算すれば安定。
ステップ2: ゲーム状態と更新処理の分離

推論: ゲームループ内で if (gameState === 'playing') のみ座標更新や衝突判定などのロジックを実行。それ以外の状態では描画のみ行い、状態ごとに表示内容を切り替える。
反証的検証: ポーズ中に gameState を切り替えるだけで更新が停止されるため、無駄な処理を省け、バグも起きにくい。
ステップ3: 自機の移動範囲制限

推論: Canvasの幅・高さから自機のサイズ（三角形の外接矩形）分を考慮して移動範囲を制限。
反証的検証: 単純に Math.min / Math.max で境界をクリップすれば、はみ出し防止と画面内収納が同時に実現できる。
ステップ4: 背景スクロールのパフォーマンス

推論: 背景として横縞を描画する際、線の本数が多いと描画負荷が上がる。線の間隔を広げ、画面内に収まる本数に制限。
反証的検証: 線のオフセットを更新してループ描画する方法は軽量。線の本数が多くてもCanvasの描画は比較的高速だが、後々敵や弾が増えた時の負荷に備え、背景はシンプルに保つ。

// 弾の更新
function updateBullets() {
  for (let i = bullets.length-1; i >= 0; i--) {
    bullets[i].x += BULLET_SPEED;
    if (bullets[i].x > canvas.width) {
      bullets.splice(i,1);
    }
  }
}

// 敵の更新
function updateEnemies() {
  for (let i = enemies.length-1; i >= 0; i--) {
    enemies[i].x -= ENEMY_SPEED;
    if (enemies[i].x + enemies[i].width < 0) {
      enemies.splice(i,1);
    }
  }
}

// 衝突判定（弾 vs 敵）
function handleCollisions() {
  for (let i = bullets.length-1; i >= 0; i--) {
    const bullet = bullets[i];
    let hit = false;
    for (let j = enemies.length-1; j >= 0; j--) {
      const enemy = enemies[j];
      if (bullet.x < enemy.x + enemy.width &&
          bullet.x + BULLET_SIZE > enemy.x &&
          bullet.y < enemy.y + enemy.height &&
          bullet.y + BULLET_SIZE > enemy.y) {
        // 衝突
        hit = true;
        enemy.hp--;
        if (enemy.hp <= 0) {
          enemies.splice(j,1);
          // スコア加算（後で実装）
          console.log("敵撃破");
        }
        break;
      }
    }
    if (hit) {
      bullets.splice(i,1);
    }
  }
}

反証的検証: 矩形判定で簡単に実装できるが、自機の三角形と敵の矩形で違和感がある場合は後で円形判定などに変更可能。ライフ減少時に自機を弾く処理はなく、単にダメージを受けて敵も消える（ゲームバランス的には自機がやられるリスクが高い）。必要に応じて、衝突時に敵だけを消すか両方消すかを調整。
ステップ2: 無敵時間の実装と視覚的フィードバック

推論: player.invincibleFrames カウンタを設け、毎フレーム減少。0以上なら衝突判定をスキップ。描画時にはカウンタが奇数のフレームで半透明にするなど点滅させる。
反証的検証: 点滅は偶数フレームのみ描画することで簡単に実現できるが、フレームレートに依存するため、時間ベースの点滅にしてもよい。ここでは簡易的に Math.floor(Date.now() / 100) % 2 === 0 などでも良いが、カウンタベースで実装する。
ステップ3: ボム機能の実装と使用回数制限

推論: Xキー押下時、bombCount > 0 かつ gameState === 'playing' なら、敵配列を空にし、自機に無敵時間を付与し、bombCount を減らす。連続ボム防止のため、ボム発動中フラグは不要（すぐに複数回押しても1回だけ発動するようにキー押下で即時処理）。
反証的検証: キーリピートにより1フレームに複数回発動する可能性を避けるため、ボム処理内で bombCount を減らした後、フラグを立ててそのフレーム内での重複発動を防ぐか、あるいはキー入力のイベントリスナー内で一度だけ実行されるようにする。今回は update ループ内で keys.x フラグを見て、フラグが立っていてボム可能なら実行し、即座にフラグを下ろす方法が安全。
ステップ4: ゲームオーバーとリスタート

推論: ライフが0以下になったら gameState = 'gameOver'。Rキーで gameState = 'title' に戻し、全変数（スコア、ライフ、ボム残数、弾配列、敵配列、タイマーなど）を初期化する。
反証的検証: リスタート時に正しく初期化されないと前の状態が残るため、初期化関数 initGame() を用意し、タイトルから開始時とリスタート時に呼び出す。
ステップ5: ステージクリア条件

推論: 敵が出現し終え（enemySpawnTimer === 0 で敵が生成されなくなり）、かつ enemies.length === 0 になったら gameState = 'clear'。
反証的検証: 現状は出現タイマーが一定間隔で無限に敵を生成するため、クリア条件に到達しない。そこで、出現回数制限（例：出現上限10体）を設けるか、出現タイマーをあるフレーム数で停止させる必要がある。ここでは、敵の出現上限を設定し、上限に達したら enemySpawnTimer を -1 にして出現停止とする。クリア時はスコア表示後、Rキーでリスタート。
ステップ6: スコア加算と表示

推論: 敵撃破時に score += 10 など。draw 関数内で ctx.fillText('SCORE: '+score, 10, 30) と表示。
反証的検証: スコア表示はゲーム状態に関わらず常に表示する（タイトル画面除く）。画面端に表示。

3. 推論ステップと反証的検証
ステップ1: エディタとゲームのUI分離

推論: ゲームCanvasとは別のHTML要素でエディタを作成し、display: none/flex で切り替える。タイトル画面に「EDIT STAGE」ボタンを配置し、クリックでエディタ表示、ゲームループは停止またはタイトル画面のままにする。
反証的検証: エディタ表示中にゲームループが動き続けると、キー入力が競合する可能性がある。エディタ表示中は gameState を 'editor' に設定し、ゲーム更新を停止する（既存の gameState に追加）。これにより、入力も無効化できる。
ステップ2: 敵データの動的追加・編集・削除

推論: JavaScriptの配列 stageEnemies で敵データを保持し、編集フォームで新規追加・既存編集を行う。削除はリスト内の削除ボタンで配列から削除。
反証的検証: 編集フォームでどの敵を編集しているか管理するために、インデックスまたはIDが必要。単純な実装では「追加」のみにし、編集は削除して再追加で対応する。ただしユーザビリティを考慮して、編集機能を実装する場合は現在選択中の敵インデックスを保持する。
ステップ3: localStorage保存・読み込み

推論: localStorage.setItem('shooterStage', JSON.stringify(stageData)) で保存。読み込み時は存在確認し、存在すればパースして stageEnemies に反映。
反証的検証: データ構造が変更された場合の互換性。現時点では単純な構造なので、読み込み時に必要項目が存在するかチェックし、不足項目はデフォルト値で補う。
ステップ4: ファイルエクスポート・インポート

推論: エクスポートは Blob と URL.createObjectURL を使ってJSONファイルをダウンロード。インポートは <input type="file"> でファイル選択し、FileReader で読み込み、JSONパースして stageEnemies に反映。
反証的検証: 読み込んだJSONが不正な形式の場合、エラー処理を行い、ユーザーに通知する。try-catchでパースエラーを捕捉。
ステップ5: 出現タイミング（フレーム）の扱い

推論: 敵ごとに spawnFrame を設定し、ゲーム側ではフレームカウンタと照らし合わせて生成する。フェーズ5で実装するため、エディタでは単に数値を入力させる。

ステップ6: 形状選択とプレビュー

推論: 形状はラジオボタンまたはドロップダウンで選択。可能であれば小さなCanvasプレビューを表示すると直感的。ただし必須ではない。

3. 推論ステップと反証的検証
ステップ1: フレームカウンタによる出現スケジューリング

推論: let frameCount = 0 を導入し、毎フレーム frameCount++。currentStage の敵リストを出現フレームでソート済みとし、frameCount が次の敵の spawnFrame に達したら敵を生成し、その敵を配列から削除（またはインデックスを進める）。
反証的検証: ステージデータが大きい場合でも、先頭から順に処理するだけなので効率的。ポーズ中はフレームカウントを増やさないようにするため、gameState === 'playing' 時のみインクリメント。
ステップ2: 形状別描画の実装

推論: 敵の shape プロパティに応じて描画関数を切り替える。三角形は中心座標を基準に正三角形を描画（向きは右向き）。
反証的検証: 三角形の描画座標計算は、サイズ（外接円の半径や幅）に基づいて頂点を計算する。例えばサイズ=20の場合、中心からのオフセットを計算して頂点を求める。サイズの定義を統一する必要がある（四角は幅・高さ、円は半径、三角は外接円半径）。
ステップ3: 形状別衝突判定（弾 vs 敵）

推論: 弾は点（弾の中心）として扱い、敵の形状に応じた内外判定を行う。

    四角：矩形に点が含まれるか。

    円：中心からの距離が半径以下か。

    三角：点と三角形の内外判定（重心座標法や符号面積法）。
    反証的検証: 点と三角形の判定は計算コストがやや高いが、敵の数が数十程度なら問題ない。自機弾と三角形の衝突は、三角形の外接矩形で簡易判定することもできるが、正確性を求める場合は多角形判定を実装する。今回は実装の完全性を優先し、点と三角形の内外判定を実装する（関数 pointInTriangle）。

ステップ4: 自機と敵の衝突判定

推論: 自機（三角形）を外接矩形で近似し、敵の形状（四角・円・三角）と外接矩形同士の衝突で判定する。これにより実装を簡素化し、ゲームバランスで調整する。
反証的検証: 外接矩形近似では厳密な当たり判定にならないが、プレイヤーの体感上は十分なことが多い。必要なら後で形状別に精密判定に拡張できる。
ステップ5: 敵の弾発射（bulletType: "straight"）

推論: 敵オブジェクトに shotCooldown と shotTimer を持たせ、bulletType === "straight" の敵は一定フレーム間隔で自機方向に弾を発射。敵弾オブジェクト配列を新たに用意し、自機に向かって移動させる。弾と自機の衝突でライフ減少・無敵付与。
反証的検証: 敵が多くなると弾の数も増え、性能に影響が出る可能性がある。最大同時弾数を制限するか、敵の射撃間隔を調整する。弾は自機の座標を元に速度ベクトルを計算。
ステップ6: ボムパラメータとクリア条件のデータ駆動化

推論: エディタデータに含まれる bombParams（maxBombCount, invincibleFrames）と stageClearCondition（type, value）を読み込み、ゲーム開始時に反映。ボム使用時は bombParams.invincibleFrames を無敵時間に使用。クリア判定は enemiesDefeated が stageClearCondition.value に達したらクリア。
反証的検証: クリア条件の拡張性を考慮し、将来的に別の条件（特定敵撃破など）にも対応できるように、条件判定関数を分岐可能な構造にしておく。
ステップ7: 複数ステージの管理

推論: ステージデータを配列で持ち、クリア時に次のステージインデックスを進める。エディタで作成したステージを複数保存しておき、順次ロードする機能を追加する。
反証的検証: 現フェーズでは単一ステージのデータ駆動化に集中し、複数ステージはフェーズ6（統合）で拡張するか、または簡単な次のステージ読み込み処理を追加する。

function drawEnemy(enemy) {
  const { shape, size, x, y } = enemy;
  ctx.fillStyle = '#f00';
  switch(shape) {
    case 'square':
      ctx.fillRect(x - size/2, y - size/2, size, size);

    case 'circle':
      ctx.beginPath();
      ctx.arc(x, y, size/2, 0, Math.PI*2);
      ctx.fill();
      break;
    case 'triangle':
      ctx.beginPath();
      const half = size/2;
      const angle = 0; // 右向き

      const p2 = { x: x - size*0.4, y: y - half };
      const p3 = { x: x - size*0.4, y: y + half };
      ctx.moveTo(p1.x, p1.y);
      ctx.lineTo(p2.x, p2.y);
      ctx.lineTo(p3.x, p3.y);
      ctx.fill();
      break;

// 点と三角形の内外判定
function pointInTriangle(px, py, ax, ay, bx, by, cx, cy) {
  const v0x = cx - ax;
  const v0y = cy - ay;
  const v1x = bx - ax;
  const v1y = by - ay;
  const v2x = px - ax;
  const v2y = py - ay;
  const dot00 = v0x * v0x + v0y * v0y;
  const dot01 = v0x * v1x + v0y * v1y;
  const dot02 = v0x * v2x + v0y * v2y;
  const dot11 = v1x * v1x + v1y * v1y;
  const dot12 = v1x * v2x + v1y * v2y;
  const invDenom = 1 / (dot00 * dot11 - dot01 * dot01);
  const u = (dot11 * dot02 - dot01 * dot12) * invDenom;
  const v = (dot00 * dot12 - dot01 * dot02) * invDenom;
  return (u >= 0) && (v >= 0) && (u + v < 1);
}

// 弾 vs 敵の衝突（形状別）
function bulletCollidesWithEnemy(bullet, enemy) {
  const bx = bullet.x + bullet.width/2;
  const by = bullet.y + bullet.height/2;
  const ex = enemy.x;
  const ey = enemy.y;
  const size = enemy.size;
  switch(enemy.shape) {
    case 'square':
      return bx > ex - size/2 && bx < ex + size/2 && by > ey - size/2 && by < ey + size/2;
    case 'circle':
      const dx = bx - ex;
      const dy = by - ey;
      return Math.hypot(dx, dy) < size/2;
    case 'triangle':
      const half = size/2;
      const p1 = { x: ex + size*0.6, y: ey };
      const p2 = { x: ex - size*0.4, y: ey - half };
      const p3 = { x: ex - size*0.4, y: ey + half };
      return pointInTriangle(bx, by, p1.x, p1.y, p2.x, p2.y, p3.x, p3.y);
    default:
      return false;
  }
}

// 敵の射撃更新（bulletType: "straight"）

    }
  }
}

// 敵弾の更新と自機衝突
function updateEnemyBullets() {
  for (let i = enemyBullets.length-1; i >= 0; i--) {
    const eb = enemyBullets[i];
    eb.x += eb.vx;
    eb.y += eb.vy;
    if (eb.x < 0 || eb.x > canvas.width || eb.y < 0 || eb.y > canvas.height) {
      enemyBullets.splice(i,1);
      continue;
    }
    // 自機との衝突（外接矩形）
    if (player.invincibleFrames <= 0 &&
        eb.x < player.x + player.width && eb.x + eb.width > player.x &&
        eb.y < player.y + player.height && eb.y + eb.height > player.y) {
      player.lives--;
      player.invincibleFrames = 60;
      enemyBullets.splice(i,1);
      if (player.lives <= 0) gameState = 'gameOver';
    }
  }
}

// メイン更新ループ内での呼び出し追加
function update() {
  if (gameState === 'playing') {
    frameCount++;
    updatePlayer();
    updateBullets();
    updateEnemies();
    updateEnemyShooting();
    updateEnemyBullets();
    handleCollisions();      // 既存の弾vs敵、自機vs敵
    spawnEnemies();
    handleShot();
    handleBomb();
    handleInvincibility();
    checkGameOver();
    checkStageClear();
  }
  draw();
  requestAnimationFrame(update);
}

// 衝突判定関数の修正（弾 vs 敵 部分を形状別判定に置き換え）
function handleCollisions() {
  // 弾 vs 敵
  for (let i = bullets.length-1; i >= 0; i--) {
    const bullet = bullets[i];
    let hit = false;
    for (let j = enemies.length-1; j >= 0; j--) {
      const enemy = enemies[j];
      if (bulletCollidesWithEnemy(bullet, enemy)) {
        hit = true;
        enemy.hp--;
        if (enemy.hp <= 0) {
          enemies.splice(j,1);
          score += 10;
          enemiesDefeated++;
        }
        break;
      }
    }
    if (hit) bullets.splice(i,1);
  }
  // 自機 vs 敵（外接矩形簡易判定）
  if (player.invincibleFrames <= 0) {
    for (let i = enemies.length-1; i >= 0; i--) {
      const enemy = enemies[i];
      const enemyRect = { x: enemy.x - enemy.size/2, y: enemy.y - enemy.size/2, w: enemy.size, h: enemy.size };
      if (player.x < enemyRect.x + enemyRect.w && player.x + player.width > enemyRect.x &&
          player.y < enemyRect.y + enemyRect.h && player.y + player.height > enemyRect.y) {
        player.lives--;
        player.invincibleFrames = 60;
        enemies.splice(i,1);
        if (player.lives <= 0) break;
      }
    }
  }
}

// ステージクリア条件判定（データ駆動）
function checkStageClear() {
  if (clearCondition.type === 'enemiesDefeated' && enemiesDefeated >= clearCondition.value) {
    gameState = 'clear';
  }
}

// ボムパラメータの適用（エディタデータ読み込み時）
function applyStageParams(stageData) {
  if (stageData.bombParams) {
    bombParams = stageData.bombParams;
    bombCount = bombParams.maxBombCount; // ゲーム開始時に反映
  }
  if (stageData.stageClearCondition) {
    clearCondition = stageData.stageClearCondition;
  }
}

            // デフォルトで空の場合はダミー
            gameStageEnemies = [
                { shape: 'square', hp: 1, size: 20, spawnFrame: 0, bulletType: 'none' },
                { shape: 'circle', hp: 2, size: 20, spawnFrame: 60, bulletType: 'straight' },
                { shape: 'triangle', hp: 1, size: 20, spawnFrame: 120, bulletType: 'none' }
            ];
        }
        // 出現キューをソートしてコピー（破壊的変更を避けるため深いコピー）
        window.spawnQueue = JSON.parse(JSON.stringify(gameStageEnemies));
        window.spawnQueue.sort((a,b) => a.spawnFrame - b.spawnFrame);
    }

    // 敵の出現（スケジューリング）
    function spawnEnemies() {
        while (window.spawnQueue && window.spawnQueue.length > 0 && window.spawnQueue[0].spawnFrame <= frameCount) {
            const enemyData = window.spawnQueue.shift();
            enemies.push({
                ...enemyData,
                x: canvas.width,
                y: Math.random() * (canvas.height - enemyData.size),
                width: enemyData.size,
                height: enemyData.size,
                hp: enemyData.hp,
                shotTimer: 0
            });
        }
    }

    // -------- 形状別描画 --------
    function drawEnemy(e) {
        ctx.fillStyle = '#f00';
        const x = e.x, y = e.y, size = e.size;
        switch(e.shape) {
            case 'square':
                ctx.fillRect(x - size/2, y - size/2, size, size);
                break;
            case 'circle':
                ctx.beginPath();
                ctx.arc(x, y, size/2, 0, Math.PI*2);
                ctx.fill();
                break;
            case 'triangle':
                ctx.beginPath();
                const half = size/2;
                const p1 = { x: x + size*0.6, y: y };
                const p2 = { x: x - size*0.4, y: y - half };
                const p3 = { x: x - size*0.4, y: y + half };
                ctx.moveTo(p1.x, p1.y);
                ctx.lineTo(p2.x, p2.y);
                ctx.lineTo(p3.x, p3.y);
                ctx.fill();
                break;
        }
    }

    // 点と三角形の内外判定
    function pointInTriangle(px, py, ax, ay, bx, by, cx, cy) {
        const v0x = cx - ax;
        const v0y = cy - ay;
        const v1x = bx - ax;
        const v1y = by - ay;
        const v2x = px - ax;
        const v2y = py - ay;
        const dot00 = v0x * v0x + v0y * v0y;
        const dot01 = v0x * v1x + v0y * v1y;
        const dot02 = v0x * v2x + v0y * v2y;
        const dot11 = v1x * v1x + v1y * v1y;
        const dot12 = v1x * v2x + v1y * v2y;
        const invDenom = 1 / (dot00 * dot11 - dot01 * dot01);
        const u = (dot11 * dot02 - dot01 * dot12) * invDenom;
        const v = (dot00 * dot12 - dot01 * dot02) * invDenom;
        return (u >= 0) && (v >= 0) && (u + v < 1);
    }

    // 弾 vs 敵（形状別）
    function bulletCollidesWithEnemy(bullet, enemy) {
        const bx = bullet.x + bullet.width/2;
        const by = bullet.y + bullet.height/2;
        const ex = enemy.x, ey = enemy.y, size = enemy.size;
        switch(enemy.shape) {
            case 'square':
                return bx > ex - size/2 && bx < ex + size/2 && by > ey - size/2 && by < ey + size/2;
            case 'circle':
                const dx = bx - ex, dy = by - ey;
                return Math.hypot(dx, dy) < size/2;
            case 'triangle':
                const half = size/2;
                const p1 = { x: ex + size*0.6, y: ey };
                const p2 = { x: ex - size*0.4, y: ey - half };
                const p3 = { x: ex - size*0.4, y: ey + half };
                return pointInTriangle(bx, by, p1.x, p1.y, p2.x, p2.y, p3.x, p3.y);
            default:
                return false;
        }
    }

    // 敵の射撃
    function updateEnemyShooting() {
        for (let enemy of enemies) {
            if (enemy.bulletType === 'straight') {
                if (enemy.shotTimer <= 0) {
                    const dx = player.x - enemy.x;
                    const dy = player.y - enemy.y;
                    const len = Math.hypot(dx, dy);
                    if (len > 0) {
                        const vx = dx / len * 5;
                        const vy = dy / len * 5;
                        enemyBullets.push({
                            x: enemy.x, y: enemy.y, vx: vx, vy: vy,
                            width: 5, height: 5
                        });
                    }
                    enemy.shotTimer = 60;
                } else {
                    enemy.shotTimer--;
                }
            }
        }
    }

    // 敵弾更新・自機衝突
    function updateEnemyBullets() {
        for (let i = enemyBullets.length-1; i >= 0; i--) {
            const eb = enemyBullets[i];
            eb.x += eb.vx;
            eb.y += eb.vy;
            if (eb.x < 0 || eb.x > canvas.width || eb.y < 0 || eb.y > canvas.height) {
                enemyBullets.splice(i,1);
                continue;
            }
            if (player.invincibleFrames <= 0 &&
                eb.x < player.x + player.width && eb.x + eb.width > player.x &&
                eb.y < player.y + player.height && eb.y + eb.height > player.y) {
                player.lives--;
                player.invincibleFrames = 60;
                enemyBullets.splice(i,1);
                if (player.lives <= 0) gameState = 'gameOver';
            }
        }
    }

    // 自機更新
    function updatePlayer() {
        let dx = 0, dy = 0;
        if (keys.ArrowUp) dy -= 1;
        if (keys.ArrowDown) dy += 1;
        if (keys.ArrowLeft) dx -= 1;
        if (keys.ArrowRight) dx += 1;
        if (dx !== 0 || dy !== 0) {
            const len = Math.hypot(dx, dy);
            dx /= len; dy /= len;
        }
        player.x += dx * player.speed;
        player.y += dy * player.speed;
        player.x = Math.min(Math.max(player.x, player.width/2), canvas.width - player.width/2);
        player.y = Math.min(Math.max(player.y, player.height/2), canvas.height - player.height/2);
    }

    // 弾更新
    function updateBullets() {
        for (let i=bullets.length-1; i>=0; i--) {
            bullets[i].x += BULLET_SPEED;
            if (bullets[i].x > canvas.width) bullets.splice(i,1);
        }
    }

    // 敵更新
    function updateEnemies() {
        for (let i=enemies.length-1; i>=0; i--) {
            enemies[i].x -= 3;
            if (enemies[i].x + enemies[i].size/2 < 0) enemies.splice(i,1);
        }
    }

    // 衝突判定（弾vs敵、自機vs敵）
    function handleCollisions() {
        // 弾 vs 敵
        for (let i=bullets.length-1; i>=0; i--) {
            const bullet = bullets[i];
            let hit = false;
            for (let j=enemies.length-1; j>=0; j--) {
                const enemy = enemies[j];
                if (bulletCollidesWithEnemy(bullet, enemy)) {
                    hit = true;
                    enemy.hp--;
                    if (enemy.hp <= 0) {
                        enemies.splice(j,1);
                        score += 10;
                        enemiesDefeated++;
                    }
                    break;
                }
            }
            if (hit) bullets.splice(i,1);
        }
        // 自機 vs 敵（外接矩形簡易）
        if (player.invincibleFrames <= 0) {
            for (let i=enemies.length-1; i>=0; i--) {
                const enemy = enemies[i];
                const er = { x: enemy.x - enemy.size/2, y: enemy.y - enemy.size/2, w: enemy.size, h: enemy.size };
                if (player.x < er.x + er.w && player.x + player.width > er.x &&
                    player.y < er.y + er.h && player.y + player.height > er.y) {
                    player.lives--;
                    player.invincibleFrames = 60;
                    enemies.splice(i,1);
                    if (player.lives <= 0) gameState = 'gameOver';
                    break;
                }
            }
        }
    }

    // 射撃
    function handleShot() {
        if (shotCooldown > 0) shotCooldown--;
        if (keys.z && shotCooldown === 0 && gameState === 'playing') {
            bullets.push({
                x: player.x + player.width/2 - BULLET_SIZE/2,
                y: player.y + player.height/2 - BULLET_SIZE/2,
                width: BULLET_SIZE, height: BULLET_SIZE
            });
            shotCooldown = SHOT_COOLDOWN_MAX;
        }
    }

    // ボム
    function handleBomb() {
        if (bombRequested && bombCount > 0 && gameState === 'playing') {
            enemies = [];
            enemyBullets = [];
            player.invincibleFrames = (window.bombParams && window.bombParams.invincibleFrames) ? window.bombParams.invincibleFrames : 60;
            bombCount--;
            bombRequested = false;
        }
        keys.x = false; // フラグリセット
    }

    // 無敵減少
    function handleInvincibility() {
        if (player.invincibleFrames > 0) player.invincibleFrames--;
    }

    function draw() {

3. 推論ステップと反証的検証
ステップ1: 速度係数の導入と影響範囲

推論: すべての速度関連の値に timeScale を乗算する。フレームカウントも timeScale 分だけ増やすことで、出現タイミングのスケーリングも実現できる。
反証的検証: 出現フレームの比較は spawnQueue[0].spawnFrame <= frameCount で行っている。frameCount を浮動小数にすると問題ないが、整数のままで比較すると微細な誤差が生じる可能性がある。そこで frameCount は浮動小数とし、出現判定も浮動小数で行う。描画などには影響しない。
ステップ2: SHIFTキーの押下検出とスローの有効化

推論: keydown で keys.Shift = true、keyup で false にする。update ループ内で timeScale = (keys.Shift && gameState === 'playing') ? 0.5 : 1.0 と設定。
反証的検証: SHIFTキーを押したままゲームオーバーやポーズになっても、状態が変われば自動的に timeScale が戻る。ポーズ中にSHIFTを押しても効果はない。
ステップ3: クールタイムと無敵フレームのスケーリング

推論: 射撃クールタイム shotCooldown は毎フレーム timeScale 分だけ減らす。敵の shotTimer も同様。自機の無敵フレーム player.invincibleFrames も timeScale 分減らす。
反証的検証: 減少量が timeScale に比例するため、スロー中はクールタイムが長く感じられる（実際の時間経過と同じ）。これで自然なスローになる。
ステップ4: ポーズからのタイトル戻し

推論: T キー押下時に gameState === 'pause' なら gameState = 'title' に変更。エディタ表示があれば非表示にするなど。
反証的検証: タイトルに戻った後、ゲーム変数は以前のまま残るが、Z キーで新規開始時に initGame() が呼ばれるため問題ない。必要なら明示的にリセットしても良いが、現状の Z キー開始処理で initGame() が呼ばれるので大丈夫。
ステップ5: キーコンフリクトの回避

推論: SHIFTキーは通常、他のキーと同時に押されることが想定される。keys.Shift フラグは独立して管理し、他のキーイベントを妨げない。
反証的検証: ブラウザによってはSHIFT+Zなどで別の動作（日本語入力など）が発生する可能性があるが、ゲーム内では特に問題ない。

// -------- キーフラグに Shift 追加 --------
const keys = {
    ArrowUp: false, ArrowDown: false, ArrowLeft: false, ArrowRight: false,
    z: false, x: false, q: false,
    Shift: false               // 追加
};

// -------- キーイベント修正 --------
function handleKeyDown(e) {
    const key = e.key;
    if (key === 'z' || key === 'Z') keys.z = true;
    if (key === 'x' || key === 'X') {
        if (gameState === 'playing') bombRequested = true;
    }
    if (key === 'q' || key === 'Q') {
        if (gameState === 'playing') gameState = 'pause';
        else if (gameState === 'pause') gameState = 'playing';
        return;
    }
    // 追加: ポーズ中にTキーでタイトル戻し
    if ((key === 't' || key === 'T') && gameState === 'pause') {
        gameState = 'title';
        // エディタが表示されていたら閉じる（念のため）
        if (document.getElementById('editorPanel').style.display === 'block') {
            document.getElementById('editorPanel').style.display = 'none';
        }
        return;
    }
    if (key === 'ArrowUp') keys.ArrowUp = true;
    if (key === 'ArrowDown') keys.ArrowDown = true;
    if (key === 'ArrowLeft') keys.ArrowLeft = true;
    if (key === 'ArrowRight') keys.ArrowRight = true;
    if (key === 'Shift') keys.Shift = true;   // SHIFT追加
    // ... その他のキー処理（R, E など）は既存のまま
}
function handleKeyUp(e) {
    const key = e.key;
    if (key === 'z' || key === 'Z') keys.z = false;
    if (key === 'x' || key === 'X') keys.x = false;
    if (key === 'ArrowUp') keys.ArrowUp = false;
    if (key === 'ArrowDown') keys.ArrowDown = false;
    if (key === 'ArrowLeft') keys.ArrowLeft = false;
    if (key === 'ArrowRight') keys.ArrowRight = false;
    if (key === 'Shift') keys.Shift = false;   // SHIFT追加
}

// -------- メインループ内で timeScale 更新 --------
function update() {
    // timeScale の更新（playing 時のみ、ポーズ中は前回の値を維持しないように）
    if (gameState === 'playing') {
        timeScale = keys.Shift ? 0.5 : 1.0;
    } else {
        timeScale = 1.0;  // 非プレイ中は通常速度（影響なし）
    }

    if (gameState === 'playing') {
        // フレームカウントに timeScale を乗算
        frameCount += timeScale;

        updatePlayer();          // 内部で速度に timeScale 適用
        updateBullets();         // 同上
        updateEnemies();         // 同上
        updateEnemyShooting();   // 内部のクールタイム更新に timeScale 適用
        updateEnemyBullets();    // 弾速度に timeScale 適用
        handleCollisions();      // 衝突判定は速度に依存しない
        spawnEnemies();          // frameCount は既に timeScale 適用済み
        handleShot();            // 内部のクールタイム更新に timeScale 適用
        handleBomb();            // ボムは即時処理、無敵フレーム減少に timeScale 適用
        handleInvincibility();   // 無敵フレーム減少に timeScale 適用
        checkGameOver();
        checkStageClear();
    }
    draw();
    requestAnimationFrame(update);
}

// -------- 各更新関数の修正（抜粋）--------
function updatePlayer() {
    let dx = 0, dy = 0;
    if (keys.ArrowUp) dy -= 1;
    if (keys.ArrowDown) dy += 1;
    if (keys.ArrowLeft) dx -= 1;
    if (keys.ArrowRight) dx += 1;
    if (dx !== 0 || dy !== 0) {
        const len = Math.hypot(dx, dy);
        dx /= len; dy /= len;
    }
    // 速度に timeScale を乗算
    player.x += dx * player.speed * timeScale;
    player.y += dy * player.speed * timeScale;
    // 範囲制限
    player.x = Math.min(Math.max(player.x, player.width/2), canvas.width - player.width/2);
    player.y = Math.min(Math.max(player.y, player.height/2), canvas.height - player.height/2);
}

function updateBullets() {
    for (let i=bullets.length-1; i>=0; i--) {
        bullets[i].x += BULLET_SPEED * timeScale;
        if (bullets[i].x > canvas.width) bullets.splice(i,1);
    }
}

function updateEnemies() {
    const ENEMY_BASE_SPEED = 3;
    for (let i=enemies.length-1; i>=0; i--) {
        enemies[i].x -= ENEMY_BASE_SPEED * timeScale;
        if (enemies[i].x + enemies[i].size/2 < 0) enemies.splice(i,1);
    }
}

function updateEnemyShooting() {
    for (let enemy of enemies) {
        if (enemy.bulletType === 'straight') {
            if (enemy.shotTimer <= 0) {
                // 弾発射処理（変わらず）
                const dx = player.x - enemy.x;
                const dy = player.y - enemy.y;
                const len = Math.hypot(dx, dy);
                if (len > 0) {
                    const vx = dx / len * 5;
                    const vy = dy / len * 5;
                    enemyBullets.push({
                        x: enemy.x, y: enemy.y, vx: vx, vy: vy,
                        width: 5, height: 5
                    });
                }
                enemy.shotTimer = 60; // ベース60フレーム
            } else {
                // クールタイム減少に timeScale 適用
                enemy.shotTimer -= timeScale;
            }
        }
    }
}

function updateEnemyBullets() {
    for (let i = enemyBullets.length-1; i >= 0; i--) {
        const eb = enemyBullets[i];
        eb.x += eb.vx * timeScale;
        eb.y += eb.vy * timeScale;
        if (eb.x < 0 || eb.x > canvas.width || eb.y < 0 || eb.y > canvas.height) {
            enemyBullets.splice(i,1);
            continue;
        }
        // 自機との衝突（変わらず）
        if (player.invincibleFrames <= 0 &&
            eb.x < player.x + player.width && eb.x + eb.width > player.x &&
            eb.y < player.y + player.height && eb.y + eb.height > player.y) {
            player.lives--;
            player.invincibleFrames = 60;
            enemyBullets.splice(i,1);
            if (player.lives <= 0) gameState = 'gameOver';
        }
    }
}

function handleShot() {
    if (shotCooldown > 0) shotCooldown -= timeScale;
    if (keys.z && shotCooldown <= 0 && gameState === 'playing') {
        bullets.push({
            x: player.x + player.width/2 - BULLET_SIZE/2,
            y: player.y + player.height/2 - BULLET_SIZE/2,
            width: BULLET_SIZE, height: BULLET_SIZE
        });
        shotCooldown = SHOT_COOLDOWN_MAX;
    }
}

function handleInvincibility() {
    if (player.invincibleFrames > 0) {
        player.invincibleFrames -= timeScale;
        if (player.invincibleFrames < 0) player.invincibleFrames = 0;
    }
}

推論: 各弾オブジェクトに trail 配列（過去座標）を追加。毎フレーム現在位置を配列にプッシュし、古いものから削除（最大5件）。描画時に過去位置を半透明（globalAlpha）で描画。
反証的検証: トレイル用配列の管理はメモリ消費が少し増えるが、弾の数は多くても数十個程度なので問題ない。globalAlpha の設定は描画順に注意（背景の上に重ねて描画）。
ステップ3: 敵弾の形状とアニメーション

推論: 敵弾を菱形（正方形を45度回転）で描画。回転角度を frameCount に応じて変化させる。サイズも脈動させる。
反証的検証: ctx.save() / ctx.restore() で回転を適用。衝突判定は矩形のまま維持するため、回転していても当たり判定は元の軸 aligned rectangle で行う。これにより実装が簡単で、プレイヤーも違和感を感じにくい。
ステップ4: 自機弾と敵弾の色と発光強化

推論: 自機弾は hsl(45, 100%, 60%) から hsl(30, 100%, 50%) へのグラデーション。敵弾は hsl(320, 100%, 60%) から hsl(340, 100%, 50%)。shadowBlur は 8 程度に設定。
反証的検証: 発光効果は描画負荷が高いが、弾の数が増えてもせいぜい数十個なので許容範囲。背景や敵の発光と重なっても視認性が向上する。
ステップ5: 衝突判定の調整

推論: 自機弾の当たり判定を円形に変更するため、bulletCollidesWithEnemy 関数内で弾の種類（自機弾か敵弾か）を判別する。自機弾は bullet.radius を持ち、円形判定。敵弾は矩形判定のまま。
反証的検証: 自機弾の radius は脈動するが、衝突判定にはベースの半径（固定）を使うことで、当たり判定が不安定になるのを防ぐ。
ステップ6: パフォーマンス最適化

推論: トレイル描画で globalAlpha を使うと、その後の描画に影響するため、必ず ctx.globalAlpha = 1 に戻す。アニメーション用の frameCount はゲーループのフレームカウンタを流用。
反証的検証: 弾のトレイル描画を毎フレーム行うと、弾の数×トレイル数の描画が発生するが、数が少ないため問題なし。

function bulletCollidesWithEnemy(bullet, enemy) {
    if (bullet.radius) { // 自機弾（円形）
        const bx = bullet.x + bullet.width/2;
        const by = bullet.y + bullet.height/2;
        const ex = enemy.x, ey = enemy.y, size = enemy.size;
        switch(enemy.shape) {
            case 'square':
                return bx > ex - size/2 && bx < ex + size/2 && by > ey - size/2 && by < ey + size/2;
            case 'circle':
                const dx = bx - ex, dy = by - ey;
                return Math.hypot(dx, dy) < size/2;
            case 'triangle':
                const half = size/2;
                const p1 = { x: ex + size*0.6, y: ey };
                const p2 = { x: ex - size*0.4, y: ey - half };
                const p3 = { x: ex - size*0.4, y: ey + half };
                return pointInTriangle(bx, by, p1.x, p1.y, p2.x, p2.y, p3.x, p3.y);
            default: return false;
        }
    } else { // 敵弾（矩形）は従来通り
        return bullet.x < enemy.x + enemy.width &&
               bullet.x + bullet.width > enemy.x &&
               bullet.y < enemy.y + enemy.height &&
               bullet.y + bullet.height > enemy.y;
    }
}

    // -------- 自機 --------
    const player = {
        x: canvas.width/2,
        y: canvas.height/2,
        width: 20,
        height: 20,
        speed: 5,
        lives: 3,
        invincibleFrames: 0
    };

    // -------- 動的オブジェクト --------
    let bullets = [];
    let enemies = [];
    let enemyBullets = [];

    // -------- 射撃関連 --------
    let shotCooldown = 0;
    const SHOT_COOLDOWN_MAX = 10;
    const BULLET_SIZE = 8;            // 拡大
    const BULLET_RADIUS = BULLET_SIZE / 2;
    const BULLET_SPEED = 8;

    // -------- キー入力 --------
    const keys = {
        ArrowUp: false, ArrowDown: false, ArrowLeft: false, ArrowRight: false,
        z: false, x: false, q: false,
        Shift: false
    };
    let bombRequested = false;

    // -------- 背景星 --------
    const stars = [];
    const STAR_COUNT = 30;
    for (let i = 0; i < STAR_COUNT; i++) {
        stars.push({
            x: Math.random() * canvas.width,
            y: Math.random() * canvas.height,
            size: 2 + Math.random() * 3,
            color: `hsl(${Math.random() * 360}, 100%, 70%)`,
            speed: 0.5 + Math.random() * 1.5
        });
    }

    // -------- ステージデータ管理 --------
    let stages = [];
    let currentStageIndex = 0;
    let currentStageEnemies = [];
    let gameStageEnemies = [];
    let bombParams = { maxBombCount: 3, invincibleFrames: 60 };
    let clearCondition = { type: 'enemiesDefeated', value: 10 };

    function getDefaultStage() {
        return {
            name: "Stage 1",
            enemies: [
                { shape: 'square', hp: 1, size: 20, spawnFrame: 0, bulletType: 'none' },
                { shape: 'circle', hp: 2, size: 20, spawnFrame: 60, bulletType: 'straight' },
                { shape: 'triangle', hp: 1, size: 20, spawnFrame: 120, bulletType: 'none' }
            ],
            bombParams: { maxBombCount: 3, invincibleFrames: 60 },
            clearCondition: { type: 'enemiesDefeated', value: 3 }
        };
    }

    function initStages() {
        stages = [getDefaultStage()];
        currentStageIndex = 0;
        currentStageEnemies = JSON.parse(JSON.stringify(stages[0].enemies));
        updateStageSelect();
    }

    function updateStageSelect() {
        const select = document.getElementById('stageSelect');
        select.innerHTML = '';
        stages.forEach((stage, idx) => {
            const option = document.createElement('option');
            option.value = idx;
            option.textContent = stage.name;
            if (idx === currentStageIndex) option.selected = true;
            select.appendChild(option);
        });
    }

    function loadStageFromSelect() {
        const select = document.getElementById('stageSelect');
        const idx = parseInt(select.value);
        if (!isNaN(idx) && stages[idx]) {
            currentStageIndex = idx;
            currentStageEnemies = JSON.parse(JSON.stringify(stages[idx].enemies));
            renderEnemyList();
        }
    }

    function newStage() {
        const name = prompt("Stage name:", "New Stage");
        if (name) {
            stages.push({
                name: name,
                enemies: [],
                bombParams: { maxBombCount: 3, invincibleFrames: 60 },
                clearCondition: { type: 'enemiesDefeated', value: 10 }
            });
            currentStageIndex = stages.length - 1;
            currentStageEnemies = [];
            updateStageSelect();
            renderEnemyList();
        }
    }

    function deleteStage() {
        if (stages.length <= 1) {
            alert("Cannot delete the last stage.");
            return;
        }
        stages.splice(currentStageIndex, 1);
        if (currentStageIndex >= stages.length) currentStageIndex = stages.length - 1;
        currentStageEnemies = JSON.parse(JSON.stringify(stages[currentStageIndex].enemies));
        updateStageSelect();
        renderEnemyList();
    }

    function saveCurrentStage() {
        stages[currentStageIndex].enemies = JSON.parse(JSON.stringify(currentStageEnemies));
        alert("Stage saved: " + stages[currentStageIndex].name);
    }

    function renderEnemyList() {
        const list = document.getElementById('enemyList');
        list.innerHTML = '';
        currentStageEnemies.forEach((enemy, idx) => {
            const li = document.createElement('li');
            li.textContent = `${enemy.shape} (HP:${enemy.hp}, Size:${enemy.size}, Frame:${enemy.spawnFrame}, Bullet:${enemy.bulletType})`;
            const delBtn = document.createElement('button');
            delBtn.textContent = 'Del';
            delBtn.onclick = () => {
                currentStageEnemies.splice(idx, 1);
                renderEnemyList();
            };
            li.appendChild(delBtn);
            list.appendChild(li);
        });
    }

    function addEnemy() {
        const shape = document.getElementById('shape').value;
        const hp = parseInt(document.getElementById('hp').value);
        const size = parseInt(document.getElementById('size').value);
        const spawnFrame = parseInt(document.getElementById('spawnFrame').value);
        const bulletType = document.getElementById('bulletType').value;
        currentStageEnemies.push({ shape, hp, size, spawnFrame, bulletType });
        currentStageEnemies.sort((a,b) => a.spawnFrame - b.spawnFrame);
        renderEnemyList();
    }

    function applyToGame() {
        const stage = stages[currentStageIndex];
        gameStageEnemies = JSON.parse(JSON.stringify(stage.enemies));
        gameStageEnemies.sort((a,b) => a.spawnFrame - b.spawnFrame);
        bombParams = stage.bombParams;
        clearCondition = stage.clearCondition;
        alert(`Applied stage "${stage.name}" to game. Start game to play.`);
    }

    function saveToLocalStorage() {
        localStorage.setItem('shooterStages', JSON.stringify(stages));
        alert('All stages saved to localStorage');
    }

    function loadFromLocalStorage() {
        const data = localStorage.getItem('shooterStages');
        if (data) {
            try {
                stages = JSON.parse(data);
                if (!stages.length) throw new Error();
                currentStageIndex = 0;
                currentStageEnemies = JSON.parse(JSON.stringify(stages[0].enemies));
                updateStageSelect();
                renderEnemyList();
                alert('Loaded from localStorage');
            } catch(e) {
                alert('Invalid data');
            }
        } else {
            alert('No saved data');
        }
    }

    function exportJSON() {
        const jsonStr = JSON.stringify(stages, null, 2);
        const blob = new Blob([jsonStr], {type: 'application/json'});
        const url = URL.createObjectURL(blob);
        const a = document.createElement('a');
        a.href = url;
        a.download = 'shooter_stages.json';
        a.click();
        URL.revokeObjectURL(url);
    }

    function importJSON(file) {
        const reader = new FileReader();
        reader.onload = (ev) => {
            try {
                const imported = JSON.parse(ev.target.result);
                if (Array.isArray(imported) && imported.length) {
                    stages = imported;
                    currentStageIndex = 0;
                    currentStageEnemies = JSON.parse(JSON.stringify(stages[0].enemies));
                    updateStageSelect();
                    renderEnemyList();
                    alert('Imported stages');
                } else {
                    throw new Error();
                }
            } catch(e) {
                alert('Invalid JSON file');
            }
        };
        reader.readAsText(file);
    }

    // -------- ゲーム初期化 --------
    function initGame() {
        score = 0;
        bombCount = bombParams.maxBombCount;
        enemiesDefeated = 0;
        bullets = [];
        enemies = [];
        enemyBullets = [];
        player.lives = 3;
        player.invincibleFrames = 0;
        player.x = canvas.width/2;
        player.y = canvas.height/2;
        shotCooldown = 0;
        frameCount = 0;
        if (!gameStageEnemies || gameStageEnemies.length === 0) {
            gameStageEnemies = [
                { shape: 'square', hp: 1, size: 20, spawnFrame: 0, bulletType: 'none', maxHp: 1 },
                { shape: 'circle', hp: 2, size: 20, spawnFrame: 60, bulletType: 'straight', maxHp: 2 },
                { shape: 'triangle', hp: 1, size: 20, spawnFrame: 120, bulletType: 'none', maxHp: 1 }
            ];
        }
        window.spawnQueue = JSON.parse(JSON.stringify(gameStageEnemies));
        window.spawnQueue.sort((a,b) => a.spawnFrame - b.spawnFrame);
    }

    // -------- ゲームロジック（フェーズ1～3統合）--------
    function spawnEnemies() {
        while (window.spawnQueue && window.spawnQueue.length > 0 && window.spawnQueue[0].spawnFrame <= frameCount) {
            const enemyData = window.spawnQueue.shift();
            enemies.push({
                ...enemyData,
                x: canvas.width,
                y: Math.random() * (canvas.height - enemyData.size),
                width: enemyData.size,
                height: enemyData.size,
                hp: enemyData.hp,
                maxHp: enemyData.hp,
                shotTimer: 0
            });
        }
    }

    function drawEnemy(e) {
        const hpRatio = e.hp / e.maxHp;
        let baseHue;
        switch(e.shape) {
            case 'square': baseHue = 0; break;
            case 'circle': baseHue = 200; break;
            case 'triangle': baseHue = 120; break;
            default: baseHue = 0;
        }
        const lightness = 30 + hpRatio * 50;
        ctx.fillStyle = `hsl(${baseHue}, 100%, ${lightness}%)`;
        const x = e.x, y = e.y, size = e.size;
        switch(e.shape) {
            case 'square':
                ctx.fillRect(x - size/2, y - size/2, size, size);
                break;
            case 'circle':
                ctx.beginPath();
                ctx.arc(x, y, size/2, 0, Math.PI*2);
                ctx.fill();
                break;
            case 'triangle':
                ctx.beginPath();
                const half = size/2;
                const p1 = { x: x + size*0.6, y: y };
                const p2 = { x: x - size*0.4, y: y - half };
                const p3 = { x: x - size*0.4, y: y + half };
                ctx.moveTo(p1.x, p1.y);
                ctx.lineTo(p2.x, p2.y);
                ctx.lineTo(p3.x, p3.y);
                ctx.fill();
                break;
        }
    }

    function pointInTriangle(px, py, ax, ay, bx, by, cx, cy) {
        const v0x = cx - ax;
        const v0y = cy - ay;
        const v1x = bx - ax;
        const v1y = by - ay;
        const v2x = px - ax;
        const v2y = py - ay;
        const dot00 = v0x * v0x + v0y * v0y;
        const dot01 = v0x * v1x + v0y * v1y;
        const dot02 = v0x * v2x + v0y * v2y;
        const dot11 = v1x * v1x + v1y * v1y;
        const dot12 = v1x * v2x + v1y * v2y;
        const invDenom = 1 / (dot00 * dot11 - dot01 * dot01);
        const u = (dot11 * dot02 - dot01 * dot12) * invDenom;
        const v = (dot00 * dot12 - dot01 * dot02) * invDenom;
        return (u >= 0) && (v >= 0) && (u + v < 1);
    }

    function bulletCollidesWithEnemy(bullet, enemy) {
        if (bullet.radius) { // 自機弾（円形）
            const bx = bullet.x + bullet.width/2;
            const by = bullet.y + bullet.height/2;
            const ex = enemy.x, ey = enemy.y, size = enemy.size;
            switch(enemy.shape) {
                case 'square':
                    return bx > ex - size/2 && bx < ex + size/2 && by > ey - size/2 && by < ey + size/2;
                case 'circle':
                    const dx = bx - ex, dy = by - ey;
                    return Math.hypot(dx, dy) < size/2;
                case 'triangle':
                    const half = size/2;
                    const p1 = { x: ex + size*0.6, y: ey };
                    const p2 = { x: ex - size*0.4, y: ey - half };
                    const p3 = { x: ex - size*0.4, y: ey + half };
                    return pointInTriangle(bx, by, p1.x, p1.y, p2.x, p2.y, p3.x, p3.y);
                default: return false;
            }
        } else { // 敵弾（矩形）
            return bullet.x < enemy.x + enemy.width &&
                   bullet.x + bullet.width > enemy.x &&
                   bullet.y < enemy.y + enemy.height &&
                   bullet.y + bullet.height > enemy.y;
        }
    }

    function updatePlayer() {
        let dx = 0, dy = 0;
        if (keys.ArrowUp) dy -= 1;
        if (keys.ArrowDown) dy += 1;
        if (keys.ArrowLeft) dx -= 1;
        if (keys.ArrowRight) dx += 1;
        if (dx !== 0 || dy !== 0) {
            const len = Math.hypot(dx, dy);
            dx /= len; dy /= len;
        }
        player.x += dx * player.speed * timeScale;
        player.y += dy * player.speed * timeScale;
        player.x = Math.min(Math.max(player.x, player.width/2), canvas.width - player.width/2);
        player.y = Math.min(Math.max(player.y, player.height/2), canvas.height - player.height/2);
    }

    function updateBullets() {
        for (let i = bullets.length-1; i >= 0; i--) {
            const b = bullets[i];
            b.x += BULLET_SPEED * timeScale;
            b.trail.push({ x: b.x, y: b.y });
            if (b.trail.length > 3) b.trail.shift();
            if (b.x > canvas.width) bullets.splice(i,1);
        }
    }

    function updateEnemies() {
        const ENEMY_BASE_SPEED = 3;
        for (let i = enemies.length-1; i >= 0; i--) {
            enemies[i].x -= ENEMY_BASE_SPEED * timeScale;
            if (enemies[i].x + enemies[i].size/2 < 0) enemies.splice(i,1);
        }
    }

    function updateEnemyShooting() {
        for (let enemy of enemies) {
            if (enemy.bulletType === 'straight') {
                if (enemy.shotTimer <= 0) {
                    const dx = player.x - enemy.x;
                    const dy = player.y - enemy.y;
                    const len = Math.hypot(dx, dy);
                    if (len > 0) {
                        const vx = dx / len * 5;
                        const vy = dy / len * 5;
                        enemyBullets.push({
                            x: enemy.x, y: enemy.y, vx: vx, vy: vy,
                            width: 8, height: 8,
                            trail: [],
                            angle: 0
                        });
                    }
                    enemy.shotTimer = 60;
                } else {
                    enemy.shotTimer -= timeScale;
                }
            }
        }
    }

    function updateEnemyBullets() {
        for (let i = enemyBullets.length-1; i >= 0; i--) {
            const eb = enemyBullets[i];
            eb.x += eb.vx * timeScale;
            eb.y += eb.vy * timeScale;
            eb.trail.push({ x: eb.x, y: eb.y });
            if (eb.trail.length > 3) eb.trail.shift();
            eb.angle = (eb.angle || 0) + 0.2 * timeScale;
            if (eb.x < 0 || eb.x > canvas.width || eb.y < 0 || eb.y > canvas.height) {
                enemyBullets.splice(i,1);
                continue;
            }
            if (player.invincibleFrames <= 0 &&
                eb.x < player.x + player.width && eb.x + eb.width > player.x &&
                eb.y < player.y + player.height && eb.y + eb.height > player.y) {
                player.lives--;
                player.invincibleFrames = 60;
                enemyBullets.splice(i,1);
                if (player.lives <= 0) gameState = 'gameOver';
            }
        }
    }

    function handleCollisions() {
        // 弾 vs 敵
        for (let i = bullets.length-1; i >= 0; i--) {
            const bullet = bullets[i];
            let hit = false;
            for (let j = enemies.length-1; j >= 0; j--) {
                const enemy = enemies[j];
                if (bulletCollidesWithEnemy(bullet, enemy)) {
                    hit = true;
                    enemy.hp--;
                    if (enemy.hp <= 0) {
                        enemies.splice(j,1);
                        score += 10;
                        enemiesDefeated++;
                    }
                    break;
                }
            }
            if (hit) bullets.splice(i,1);
        }
        // 自機 vs 敵（外接矩形）
        if (player.invincibleFrames <= 0) {
            for (let i = enemies.length-1; i >= 0; i--) {
                const enemy = enemies[i];
                const er = { x: enemy.x - enemy.size/2, y: enemy.y - enemy.size/2, w: enemy.size, h: enemy.size };
                if (player.x < er.x + er.w && player.x + player.width > er.x &&
                    player.y < er.y + er.h && player.y + player.height > er.y) {
                    player.lives--;
                    player.invincibleFrames = 60;
                    enemies.splice(i,1);
                    if (player.lives <= 0) gameState = 'gameOver';
                    break;
                }
            }
        }
    }

    function handleShot() {
        if (shotCooldown > 0) shotCooldown -= timeScale;
        if (keys.z && shotCooldown <= 0 && gameState === 'playing') {
            bullets.push({
                x: player.x + player.width/2 - BULLET_SIZE/2,
                y: player.y + player.height/2 - BULLET_SIZE/2,
                width: BULLET_SIZE,
                height: BULLET_SIZE,
                radius: BULLET_RADIUS,
                trail: []
            });
            shotCooldown = SHOT_COOLDOWN_MAX;
        }
    }

    function handleBomb() {
        if (bombRequested && bombCount > 0 && gameState === 'playing') {
            enemies = [];
            enemyBullets = [];
            player.invincibleFrames = bombParams.invincibleFrames;
            bombCount--;
            bombRequested = false;
        }
        keys.x = false;
    }

    function handleInvincibility() {
        if (player.invincibleFrames > 0) {
            player.invincibleFrames -= timeScale;
            if (player.invincibleFrames < 0) player.invincibleFrames = 0;
        }
    }

    function checkGameOver() {
        if (player.lives <= 0) gameState = 'gameOver';
    }

    function checkStageClear() {
        if (clearCondition.type === 'enemiesDefeated' && enemiesDefeated >= clearCondition.value) {
            gameState = 'clear';
        }
    }

    // -------- 描画（フェーズ2,3統合）--------
    function draw() {
        // 背景グラデーション
        const grad = ctx.createLinearGradient(0, 0, 0, canvas.height);
        grad.addColorStop(0, '#0a0f2a');
        grad.addColorStop(1, '#4a1a6a');
        ctx.fillStyle = grad;
        ctx.fillRect(0, 0, canvas.width, canvas.height);
        
        // 星
        ctx.shadowBlur = 0;
        for (const star of stars) {
            ctx.fillStyle = star.color;
            ctx.beginPath();
            ctx.arc(star.x, star.y, star.size, 0, Math.PI * 2);
            ctx.fill();
            star.x -= star.speed * timeScale;
            if (star.x < 0) {
                star.x = canvas.width;
                star.y = Math.random() * canvas.height;
            }
        }
        
        // 自機（虹色グラデーション）
        const hue = (Date.now() * 0.005) % 360;
        const gradPlayer = ctx.createLinearGradient(player.x - 10, player.y - 10, player.x + 10, player.y + 10);
        gradPlayer.addColorStop(0, `hsl(${hue}, 100%, 60%)`);
        gradPlayer.addColorStop(1, `hsl(${(hue + 60) % 360}, 100%, 50%)`);
        ctx.fillStyle = gradPlayer;
        ctx.shadowBlur = 10;
        ctx.shadowColor = `hsl(${hue}, 100%, 60%)`;
        if (!(player.invincibleFrames > 0 && (Math.floor(Date.now() / 50) % 2 === 0))) {
            ctx.beginPath();
            const angle = 0;
            const x1 = player.x + 12 * Math.cos(angle);
            const y1 = player.y + 12 * Math.sin(angle);
            const x2 = player.x + 8 * Math.cos(angle + 2.5);
            const y2 = player.y + 8 * Math.sin(angle + 2.5);
            const x3 = player.x + 8 * Math.cos(angle - 2.5);
            const y3 = player.y + 8 * Math.sin(angle - 2.5);
            ctx.moveTo(x1, y1);
            ctx.lineTo(x2, y2);
            ctx.lineTo(x3, y3);
            ctx.fill();
        }
        
        // 敵
        for (const e of enemies) {
            ctx.shadowBlur = 5;
            let baseHue;
            switch(e.shape) {
                case 'square': baseHue = 0; break;
                case 'circle': baseHue = 200; break;
                case 'triangle': baseHue = 120; break;
                default: baseHue = 0;
            }
            ctx.shadowColor = `hsl(${baseHue}, 100%, 50%)`;
            drawEnemy(e);
        }
        
        // 自機弾（トレイル＋脈動＋星型）
        for (const b of bullets) {
            for (let j = 0; j < b.trail.length; j++) {
                const t = b.trail[j];
                ctx.globalAlpha = (j + 1) / b.trail.length * 0.5;
                ctx.beginPath();
                ctx.arc(t.x + BULLET_SIZE/2, t.y + BULLET_SIZE/2, BULLET_RADIUS * 0.7, 0, Math.PI*2);
                ctx.fillStyle = '#ffaa66';
                ctx.fill();
            }
            ctx.globalAlpha = 1;
            const pulse = 1 + Math.sin(frameCount * 0.2) * 0.3;
            const radius = BULLET_RADIUS * pulse;
            const gradBullet = ctx.createRadialGradient(
                b.x + BULLET_SIZE/2, b.y + BULLET_SIZE/2, radius * 0.3,
                b.x + BULLET_SIZE/2, b.y + BULLET_SIZE/2, radius
            );
            gradBullet.addColorStop(0, '#ffcc44');
            gradBullet.addColorStop(1, '#ff8844');
            ctx.fillStyle = gradBullet;
            ctx.shadowBlur = 12;
            ctx.shadowColor = '#ffaa44';
            ctx.beginPath();
            ctx.arc(b.x + BULLET_SIZE/2, b.y + BULLET_SIZE/2, radius, 0, Math.PI*2);
            ctx.fill();
            ctx.beginPath();
            for (let k = 0; k < 5; k++) {
                const ang = (k * 72 - 36) * Math.PI / 180 + frameCount * 0.05;
                const sx = b.x + BULLET_SIZE/2 + Math.cos(ang) * radius * 1.2;
                const sy = b.y + BULLET_SIZE/2 + Math.sin(ang) * radius * 1.2;
                if (k === 0) ctx.moveTo(sx, sy);
                else ctx.lineTo(sx, sy);
            }
            ctx.fillStyle = '#ffaa44';
            ctx.fill();
        }
        
        // 敵弾（トレイル＋回転＋脈動）
        for (const eb of enemyBullets) {
            for (let j = 0; j < eb.trail.length; j++) {
                const t = eb.trail[j];
                ctx.globalAlpha = (j + 1) / eb.trail.length * 0.4;
                ctx.save();
                ctx.translate(t.x + eb.width/2, t.y + eb.height/2);
                ctx.rotate(eb.angle);
                ctx.fillStyle = '#ff66aa';
                ctx.fillRect(-eb.width/2, -eb.height/2, eb.width, eb.height);
                ctx.restore();
            }
            ctx.globalAlpha = 1;
            const pulse = 1 + Math.sin(frameCount * 0.25) * 0.2;
            const w = eb.width * pulse;
            const h = eb.height * pulse;
            ctx.save();
            ctx.translate(eb.x + eb.width/2, eb.y + eb.height/2);
            ctx.rotate(eb.angle);
            const gradEB = ctx.createLinearGradient(-w/2, -h/2, w/2, h/2);
            gradEB.addColorStop(0, '#ff66aa');
            gradEB.addColorStop(1, '#ff3388');
            ctx.fillStyle = gradEB;
            ctx.shadowBlur = 10;
            ctx.shadowColor = '#ff66aa';
            ctx.fillRect(-w/2, -h/2, w, h);
            ctx.restore();
        }
        ctx.shadowBlur = 0;
        ctx.globalAlpha = 1;
        
        // UI
        ctx.fillStyle = 'rgba(0,0,0,0.6)';
        ctx.fillRect(0, 0, 200, 110);
        ctx.fillStyle = '#fff';
        ctx.font = 'bold 20px monospace';
        ctx.shadowBlur = 2;
        ctx.shadowColor = '#000';
        ctx.fillText('SCORE: ' + score, 10, 30);
        ctx.fillText('LIVES: ' + player.lives, 10, 60);
        ctx.fillText('BOMB: ' + bombCount, 10, 90);
        for (let i = 0; i < player.lives; i++) {
            ctx.fillStyle = '#ff3366';
            ctx.beginPath();
            ctx.moveTo(110 + i * 25, 55);
            ctx.bezierCurveTo(105 + i * 25, 50, 100 + i * 25, 45, 110 + i * 25, 40);
            ctx.bezierCurveTo(120 + i * 25, 45, 115 + i * 25, 50, 110 + i * 25, 55);

            frameCount += timeScale;
            updatePlayer();

            updateEnemies();
            updateEnemyShooting();
            updateEnemyBullets();
            handleCollisions();

        requestAnimationFrame(update);
    }

    // -------- イベント設定 --------
    function handleKeyDown(e) {
        const key = e.key;
        if (key === 'z' || key === 'Z') keys.z = true;
        if (key === 'x' || key === 'X') {
            if (gameState === 'playing') bombRequested = true;
        }
        if (key === 'q' || key === 'Q') {
            if (gameState === 'playing') gameState = 'pause';
            else if (gameState === 'pause') gameState = 'playing';
            return;
        }
        if ((key === 't' || key === 'T') && gameState === 'pause') {
            gameState = 'title';
            if (document.getElementById('editorPanel').style.display === 'block') {
                document.getElementById('editorPanel').style.display = 'none';
            }
            return;
        }
        if (key === 'ArrowUp') keys.ArrowUp = true;
        if (key === 'ArrowDown') keys.ArrowDown = true;
        if (key === 'ArrowLeft') keys.ArrowLeft = true;
        if (key === 'ArrowRight') keys.ArrowRight = true;
        if (key === 'Shift') keys.Shift = true;
        if ((key === 'r' || key === 'R') && (gameState === 'gameOver' || gameState === 'clear')) {
            gameState = 'title';
        }
        if (key === 'e' || key === 'E') {
            if (gameState === 'title') {
                gameState = 'editor';
                document.getElementById('editorPanel').style.display = 'block';
            } else if (gameState === 'editor') {
                gameState = 'title';
                document.getElementById('editorPanel').style.display = 'none';
            }
        }
    }
    function handleKeyUp(e) {
        const key = e.key;
        if (key === 'z' || key === 'Z') keys.z = false;
        if (key === 'x' || key === 'X') keys.x = false;
        if (key === 'ArrowUp') keys.ArrowUp = false;
        if (key === 'ArrowDown') keys.ArrowDown = false;
        if (key === 'ArrowLeft') keys.ArrowLeft = false;
        if (key === 'ArrowRight') keys.ArrowRight = false;
        if (key === 'Shift') keys.Shift = false;
    }
    window.addEventListener('keydown', handleKeyDown);
    window.addEventListener('keyup', handleKeyUp);
    window.addEventListener('keydown', (e) => {
        if ((e.key === 'z' || e.key === 'Z') && gameState === 'title') {
            initGame();
            gameState = 'playing';
